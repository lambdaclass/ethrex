# LEVM (Lambda EVM)

Implementation of a simple Ethereum Virtual Machine in Rust.

## Status
Meaning:
- ✅: Implemented
- 🏗️: Work in Progress
- ❌: Work not Started yet

Features:
- Opcodes 🏗️
- Precompiles 🏗️
- Transaction validation 🏗️
- Pass all EF tests 🏗️
