## 발견된 버그 및 수정 사항

### 근본 원인 (Root Cause) — 토큰 이중 소스
`get_platform_token()`이 메모리가 비면 파일에서 다시 읽는 폴백 로직이 있었음.
로그아웃 후에도 파일에서 옛 토큰을 읽어오는 경로가 존재.

**수정**: 파일은 앱 시작(`new()`) 시에만 한 번 읽고, 이후 메모리만 사용.

### ChatView 상태 초기화
페이지 이동 시 ChatView가 unmount → 채팅 히스토리, 사용량, 로그인 정보 소멸.

**수정**: ChatView를 항상 마운트(`display: none`)하고, `isVisible` prop으로 상태 동기화.

### 설정 ↔ 채팅 로그인 동기화
설정에서 로그인/로그아웃해도 채팅 헤더에 반영 안됨.

**수정**: `isVisible` 변경 시 `refreshTokamakState()`로 AiProvider에서 최신 상태 조회.

### 설정 로그아웃 버튼 동작 안함
`window.confirm()`이 Tauri webview에서 정상 동작하지 않을 수 있고, `invoke` 에러 시 catch가 없어서 UI 갱신 안됨.

**수정**:
- `window.confirm()` → 인라인 확인 UI로 교체 (로그아웃 → "로그아웃 확인" + "취소" 버튼)
- `invoke('delete_platform_token')` 에러 처리 추가 (try/catch)
