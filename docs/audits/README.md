# Audits

Audit reports for ethrex are stored in the [`audits/`](https://github.com/lambdaclass/ethrex/tree/main/audits) directory at the repository root. Each subdirectory's name encodes the audited version and commit short hash.

To report a vulnerability, see the [Security Policy](https://github.com/lambdaclass/ethrex/blob/main/.github/SECURITY.md).
