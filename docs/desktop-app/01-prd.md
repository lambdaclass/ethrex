# PRD: Tokamak Desktop App (기획서)

## 1. 제품 비전

카카오톡과 같은 친숙한 데스크탑 메신저 형태의 애플리케이션에서,
AI와 대화하며 Tokamak Layer 2 네트워크를 운영·관리·탐색할 수 있는 올인원 플랫폼.

## 2. 핵심 가치

- **접근성**: 설치형 데스크탑 앱으로 누구나 쉽게 L2를 운영
- **AI-First**: 채팅으로 모든 기능을 수행 (명령어 학습 불필요)
- **통합 대시보드**: 각 레이어의 상태를 웹 대시보드로 한눈에 확인
- **생태계 연결**: L2 운영자들이 서로의 서비스를 공개하고 연결

## 3. 타겟 사용자

| 사용자 유형 | 설명 |
|------------|------|
| L2 운영자 | 자체 Layer 2를 배포·운영하려는 팀/개인 |
| 개발자 | L2 위에서 dApp을 개발하는 개발자 |
| 탐색자 | 다양한 L2 서비스를 둘러보고 사용해보려는 일반 사용자 |

## 4. 주요 기능

### 4.1 노드 제어 패널

- ethrex 클라이언트 시작/중지/재시작
- 프로버(Prover) 서버 시작/중지
- 시퀀서(Sequencer) 상태 관리
- 설정 파일 편집 (네트워크, 포트, 키 등)
- 로그 실시간 스트리밍

### 4.2 레이어별 대시보드 (웹뷰 탭)

각 대시보드는 웹 페이지 URL을 내장 브라우저(WebView)로 표시한다.

| 탭 이름 | 내용 | URL 예시 |
|---------|------|----------|
| L1 Dashboard | L1 체인 상태, 블록, 트랜잭션 | `http://localhost:3010` |
| L2 Dashboard | L2 체인 상태, 배치, 증명 | `http://localhost:3011` |
| Explorer | 블록 익스플로러 | `http://localhost:3012` |
| Prover | 증명 생성 상태, 비용 분석 | `http://localhost:3013` |
| Metrics | Grafana/Prometheus 메트릭 | `http://localhost:3000` |

### 4.3 AI 채팅 인터페이스

카카오톡 스타일의 채팅 UI에서 AI와 대화하며 앱을 제어한다.

**지원 AI 프로바이더:**
- Claude (Anthropic)
- GPT (OpenAI)
- Gemini (Google)

**채팅으로 수행 가능한 작업 예시:**
```
사용자: "L2 노드를 시작해줘"
AI: "L2 노드를 시작합니다. 설정을 확인해볼까요?"

사용자: "현재 배치 상태가 어때?"
AI: "마지막 배치 #1234가 10분 전에 제출되었고, 증명 생성 중입니다..."

사용자: "가스비 설정을 0.001 TON으로 바꿔줘"
AI: "가스비를 0.001 TON으로 변경했습니다. 다음 블록부터 적용됩니다."

사용자: "다른 L2 중에 DEX 서비스 있는 곳 찾아줘"
AI: "현재 등록된 L2 중 DEX를 운영하는 3개를 찾았습니다..."
```

### 4.4 AI 로그인/인증

- 각 AI 프로바이더별 API 키 입력 또는 OAuth 로그인
- 로그인 후 선택한 AI 모델로 채팅
- API 키는 로컬 키체인에 안전하게 저장
- 모델 선택 (예: Claude Opus, GPT-4, Gemini Pro)

### 4.5 L2 마켓플레이스 (Phase 2)

- 운영 중인 L2 목록을 공개/등록
- 각 L2가 제공하는 기능(DEX, NFT, Bridge 등) 카탈로그
- AI 가이드: "이 L2에서 토큰 스왑하는 방법" 등 사용법 안내
- AI끼리 서로의 L2를 탐색하고 사용할 수 있도록 API 명세 공개

### 4.6 TON 충전 페이지

- TON(네이티브 토큰) 잔액 확인
- L1 → L2 브릿지를 통한 TON 충전 (Deposit)
- L2 → L1 출금 (Withdraw)
- AI 에이전트 전용 지갑 생성 및 충전
- AI가 L2에서 트랜잭션을 실행하기 위한 가스비 충전

## 6. 비기능 요구사항

| 항목 | 요구사항 |
|------|---------|
| 플랫폼 | macOS, Windows, Linux |
| 설치 크기 | < 100MB (런타임 제외) |
| 메모리 | < 200MB (앱 자체) |
| 보안 | API 키 로컬 키체인 저장, HTTPS 통신 |
| 오프라인 | 노드 제어는 오프라인 가능, AI 채팅은 온라인 필요 |
| 업데이트 | 자동 업데이트 지원 |
